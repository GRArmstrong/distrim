Technical
=========

Requirements
------------

- Python 2.7
- Pip (Python Package Index)


Setup
-----

It's best to use a virtual environment.

Install the necessary Python packages as such:

  $ pip install -r requirements.txt


Usage
-----

Begin execution with the following command:

  $ ./run
